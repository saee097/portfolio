## 🚀 Getting Started

### 1. Install Dependencies
```bash
cd portfolio
npm install
```

### 2. Run Development Server
```bash
npm run dev
```
Visit: http://localhost:3000

### 3. Customize Content
Follow **CUSTOMIZATION_GUIDE.md** to update:
- Your name and title
- About section
- Skills
- Experience
- Projects
- Contact information
- Colors (optional)

### 4. Deploy to Vercel
```bash
npm run build
# Then deploy to Vercel (3 clicks!)
```

## 📝 Quick Customization (15 minutes)
